1. What are the different ways to select an element in the DOM?
  • getElementById() – Selects an element by ID.
  • getElementsByClassName() – Selects elements by class name.
  • getElementsByTagName() – Selects elements by tag name.
  • querySelector() – Selects the first matching CSS selector.
  • querySelectorAll() – Selects all matching elements.
2. What is the difference between innerHTML, innerText, and textContent?
  • innerHTML: Gets/sets HTML inside an element. Supports HTML tags.
  • innerText: Gets/sets visible text only. Respects CSS styling.
  • textContent: Gets/sets all text content, regardless of styling.
3. What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events triggered by its children. Instead of attaching listeners to each child, one listener on the parent captures events as they bubble up. It's efficient and ideal for dynamic content.

4. What is event bubbling in the DOM?

Event bubbling is the process where an event triggered on a child element propagates up to its parent elements. For example, clicking a button inside a div will first trigger the button’s handler, then the div’s handler. It can be stopped using event.stopPropagation().

5. How do you create, add, and remove elements using JavaScript?
  • Create: document.createElement("tag")
  • Add: parent.appendChild(newElement) or append()
  • Remove: element.remove() or parent.removeChild(child)
